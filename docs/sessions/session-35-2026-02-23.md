## Key Decisions
1. **Hook system as middleware chain** (MS Agent Framework pattern, not simple before/after hooks)
2. **ISC as `GuardrailResult(bool, Any)` interface** (CrewAI pattern)
3. **Heuristic-first effort classification** (not LLM-based, <10ms budget)
4. **Circuit breakers on all tool executions** (first in ecosystem)
5. **Agent Skills standard compatibility** for ecosystem interop
6. **4 differentiation features** no other platform has built yet
