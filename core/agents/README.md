# Agents Canon

- Canonical agent instructions live in `core/agents/`.
- Archived variants from source repos live in `core/agents/sources/` and are not authoritative.

## Primary Files
- `CLAUDE.md`: primary agent instructions (canonical).
- `CLAUDE_SESSION_WRAP_CONTEXT_AGENT.md`: session wrap guidance (canonical).
